Desafío 1 - Mi Curriculum Vitae en HTML

El objetivo del proyecto es crear una página web simple y efectiva que funcione como el Curriculum Vitae (CV) en línea de Carlos Carrillo. La página debe presentar información personal, académica y profesional de manera organizada y fácil de navegar.

Desafío 2 - Construyendo un landing page

El objetivo de este proyecto es crear una landing page para la empresa CompuTECH. La página está diseñada para mostrar de manera atractiva y clara las mejores computadoras disponibles en el mercado que ofrece la empresa. Se pretende atraer a posibles clientes proporcionando información detallada de los productos, junto con una visualización atractiva de los mismos.

Desafío 3 - Iguana page

El objetivo de este proyecto es crear una página informativa sobre iguanas, proporcionando detalles visuales y textuales que eduquen al visitante sobre este reptil. La página está diseñada para ser atractiva y fácil de navegar, destacando las características y peculiaridades de las iguanas.

Desafío 4 - Cuppon

El objetivo de este proyecto es crear una landing page para la plataforma CUPPON, que ofrece una variedad de cupones de descuento para productos y servicios. La página está diseñada para ser visualmente atractiva, fácil de navegar y optimizada para convertir visitantes en suscriptores y clientes.

Desafío 5 - Github

El objetivo de este proyecto es crear una página web de portafolio personal para Carlos Carrillo. La página está diseñada para mostrar sus habilidades, experiencia, trabajos anteriores y proporcionar un medio de contacto para posibles colaboraciones y oportunidades profesionales.
